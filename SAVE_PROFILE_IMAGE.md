# How to Add Your Profile Image

## Quick Steps:

1. **Save the uploaded image** (the anime-style portrait you just shared) to your computer

2. **Rename it** to `hero.jpg`

3. **Copy it to**: `C:\Users\kirkaur\Videos\portfolio 2\public\images\profile\hero.jpg`

4. **Copy the same file again** and rename to `about.jpg` in the same folder

## Final File Structure:

```
portfolio 2/
└── public/
    └── images/
        └── profile/
            ├── hero.jpg   (your profile image)
            └── about.jpg  (same image, duplicate)
```

## After Saving:

- The image will appear on the **homepage hero section** (large circular profile image)
- The image will appear on the **about page** (square with rounded corners)
- Both use the same image for consistency
